Welcome to the Atlas-SoC development kit.

Before you begin you will need to install some drivers. Here's how:

       On the SD Card, look for the "Drivers" folder for your host OS and install the appropriate drivers:

  • For Windows machines please install the drivers which are appropriate for your OS:

    • ATLAS_D64 (for 64-bit OS)

    • ATLAS_DRV (for 32-bit OS)

       Linux users can ignore this step.

The board is running a web server and provides a web site through which you will interact with the board. We strongly recommend using either Google Chrome or Mozilla Firefox.

  • Open your web browser, type 192.168.7.1 into the URL pane . . . or just click here.

The web site served by the kit will be your guide as you learn about the board, play with the design examples, and experience developing for Altera SoC FPGA devices. We hope the Atlas-SoC kit experience is both informative and fun. When you are ready, please tell us what you think using the on-line survey; there is a link located at the bottom of the Develop page. Your feedback will help us make the next release better!
